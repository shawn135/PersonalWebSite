Scalable E-Commerce Platform

Overview

This is an aws project to design a Scalable E-Commerce Platform using AWS services, including Lambda, API Gateway, S3, RDS, Docker, and EC2.

Details

  • Built a Python-based microservice hosted in a Docker container, integrating it with an AWS-backed infrastructure for seamless API communication.
  • Configured a highly available architecture with public and private subnets across multiple Availability Zones.
  • Leveraged IAM roles, security groups, and VPC configurations to ensure a secure and isolated environment.
  • Deployed a Dockerized application to AWS Elastic Container Service (ECS) and integrated it with an RDS MySQL database for persistent storage.
  • Automated resource provisioning using AWS CloudFormation to enable reproducibility and minimize manual setup.
  • Tested the application locally using Docker and exposed services for API testing, ensuring reliability before deployment.
